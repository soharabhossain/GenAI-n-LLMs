-------------------------------
# Hugging Face Evaluate Library
-------------------------------
The Hugging Face evaluate library is used to calculate metrics like BLEU and ROUGE for LLM evaluation. These metrics compare machine-generated text against a human-written reference text based on word or n-gram overlap.

https://huggingface.co/docs/evaluate/index
